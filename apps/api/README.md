# Corrupt Video Inspector API

REST API for video corruption detection services (placeholder).

## Overview

This is a placeholder for the future REST API that will provide programmatic access to video corruption detection capabilities.

## Planned Features

- REST endpoints for video scanning
- Job queue management
- Results persistence
- Authentication and authorization
- Integration with CLI core functionality

## Development

This application is currently a placeholder. Implementation will include:

```bash
# Start development server
nx serve api

# Run tests
nx test api

# Build for production
nx build api
```

## Dependencies

Will depend on the `core` library for shared business logic.
