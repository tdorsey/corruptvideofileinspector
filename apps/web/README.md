# Corrupt Video Inspector Web UI

React-based web interface for video corruption detection (placeholder).

## Overview

This is a placeholder for the future web UI that will provide a browser-based interface for video corruption detection.

## Planned Features

- File upload and scanning interface
- Real-time scan progress tracking
- Results visualization
- Integration with API backend
- Responsive design

## Technology Stack

- React 18+
- Vite for build tooling
- TypeScript
- Shared UI components from `libs/ui-components`

## Development

This application is currently a placeholder. Implementation will include:

```bash
# Start development server
nx serve web

# Run tests
nx test web

# Build for production
nx build web

# Preview production build
nx preview web
```

## Dependencies

Will depend on the `ui-components` library for shared React components.
